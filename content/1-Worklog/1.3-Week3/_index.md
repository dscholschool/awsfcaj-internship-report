### Week 3 Objectives:

* Practice Amazon EC2 by launching an instance and configuring network access.
* Understand Security Groups and SSH connection methods for Linux-based EC2 instances.
* Deploy a sample web application to EC2 for the first time.
* Practice IAM User, IAM Role, and Policy configuration for access control.

### Week 3 Achievements:

* Created an EC2 instance and understood the role of AMI, instance type, key pair, and storage configuration.
* Configured Security Group inbound rules for SSH and web access.
* Successfully connected to the EC2 instance using SSH and performed basic server operations.
* Deployed and tested a sample web application on EC2.
* Practiced IAM permission assignment through users, roles, and policies.
* Understood how EC2 can act as the application server in a cloud-based web system.
